# /tech-stack — 技術スタック決定スキル

## 説明
アーキテクチャ設計を踏まえて、技術スタックを確認・決定するスキル。

## 使い方
```
/tech-stack          # 現在のスタックを確認し、追加が必要か判断
/tech-stack "状態管理" # 特定領域のスタックを検討
```

## 実行手順

1. **現状把握**:
   - `CLAUDE.md` の技術スタックセクションを確認する
   - `package.json` の依存関係を確認する

2. **新規ライブラリの検討**（必要な場合）:
   - 候補を2〜3個リストアップし、比較する
   - 選定基準: バンドルサイズ、メンテナンス状況、既存スタックとの親和性
   - 選定理由を明記する

3. **ユーザー承認**: ライブラリ追加が必要な場合、理由とともにユーザーに提案する。

4. **適用**:
   - `package.json` を更新（`npm install`）
   - `CLAUDE.md` の技術スタックセクションを更新する

## 出力
- 更新された `CLAUDE.md`（必要時）
- インストール済みの依存パッケージ
