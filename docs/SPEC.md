# 機能仕様書

## サービス概要
**QiitaAI Watch** — QiitaおよびZenn上のAI関連記事を自動収集・閲覧できるWebサイト。

---

## フェーズ1（MVP）

### 画面一覧

#### 1. ホーム画面 (`/`)
- AI関連記事のカード一覧を表示する
- 初期表示は「新着順」で20件
- ページネーションで次ページを取得できる

#### 2. Aboutページ (`/about`)
- サイトの説明文を表示する静的ページ

---

### 記事カードの表示項目
| 項目 | 表示内容 |
|------|----------|
| タイトル | 記事タイトル（クリックでQiita元記事へ遷移） |
| 著者 | アイコン画像 + ユーザー名 |
| 投稿日 | YYYY/MM/DD 形式 |
| いいね数 | ハートアイコン + 数値 |
| タグ | 最大5件のタグバッジ |

---

### データソース
- **Qiita**: Qiita API v2 を直接使用
- **Zenn**: Zenn公開API (`https://zenn.dev/api/articles`) を使用

### ソースフィルター
- すべて（Qiita + Zenn の記事を統合表示）
- Qiitaのみ
- Zennのみ

---


### フィルタリング・ソート機能

#### UIレイアウト
- ホーム画面はサイドバー型レイアウトを採用する
  - 左カラム（幅: 固定 240px / md以上）: フィルターサイドバー
  - 右カラム: 記事一覧
  - md未満（モバイル）: サイドバーを上部に折りたたみ表示

#### タグフィルター
- タグ別フィルター（複数選択可）
- 対象タグ: `AI`, `機械学習`, `LLM`, `ChatGPT`, `DeepLearning`, `OpenAI`, `RAG`, `生成AI`
- 未選択時はすべてのAI関連記事を表示
- Zennはタグに対応するトピック名で検索

#### 期間フィルター
- 投稿日による絞り込み
- 選択肢: `全期間`（デフォルト）/ `今週` / `今月` / `3ヶ月以内`
- Qiita API の `created:>=YYYY-MM-DD` クエリを使用して実装

#### ソート
- 新着順（デフォルト）
- いいね数順

### ページネーション
- 1ページ20件
- 「前へ」「次へ」ボタン

---

### 非機能要件
- レスポンシブ対応（ブレークポイント: sm=640px, md=768px, lg=1024px）
- ダークモード対応（OSの設定に追従）
- ローディング中はスケルトンUIを表示
- APIエラー時はエラーメッセージを表示（リトライボタン付き）

---

## フェーズ2（拡張）

- ユーザー認証・ログイン機能
- サーバーサイドの実装（フロントエンドのみ）
- 記事の投稿・編集機能

# GitHub Trending Viewer 仕様書

## サービス概要
**GitHub Trending Viewer** — GitHub Search APIを使ってトレンドのリポジトリを自動収集・表示するWebアプリ。フロントエンド完結。

---

## フェーズ1（MVP）

### 画面一覧

#### 1. ホーム画面 (`/`)
- トレンドリポジトリのカード一覧を表示する
- 初期表示は「今週作成された Star 数順」で20件

#### 2. Aboutページ (`/about`)
- サービス説明文を表示する静的ページ

---

### リポジトリカードの表示項目
| 項目 | 表示内容 |
|------|----------|
| リポジトリ名 | オーナー / リポジトリ名（クリックでGitHubへ遷移） |
| 説明文 | リポジトリのdescription |
| 言語 | プログラミング言語バッジ |
| Star数 | ⭐ + 数値 |
| Fork数 | 🍴 + 数値 |
| オーナーアイコン | アバター画像 |
| 作成日 | YYYY/MM/DD 形式 |

---

### データソース
- **GitHub Search API**: `https://api.github.com/search/repositories`（認証不要、60req/h）

---

### フィルタリング・ソート機能

#### 期間フィルター
- 選択肢: `今日`（デフォルト） / `今週` / `今月`
- GitHub Search APIの `created:>=YYYY-MM-DD` クエリで実装

#### 言語フィルター
- 対象言語: `すべて`（デフォルト）+ 主要言語一覧
- `language:xxx` クエリで実装

### ページネーション
- 1ページ20件
- 「前へ」「次へ」ボタン

---

### 非機能要件
- レスポンシブ対応（ブレークポイント: sm=640px, md=768px）
- ダークモード対応（OSの設定に追従）
- ローディング中はスケルトンUIを表示
- APIエラー時はエラーメッセージを表示（リトライボタン付き）
