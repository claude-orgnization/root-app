# 機能仕様書

## サービス概要
**QiitaAI Watch** — QiitaおよびZenn上のAI関連記事を自動収集・閲覧できるWebサイト。

---

## フェーズ1（MVP）

### 画面一覧

#### 1. ホーム画面 (`/`)
- AI関連記事のカード一覧を表示する
- 初期表示は「新着順」で20件
- ページネーションで次ページを取得できる

#### 2. Aboutページ (`/about`)
- サイトの説明文を表示する静的ページ

---

### 記事カードの表示項目
| 項目 | 表示内容 |
|------|----------|
| タイトル | 記事タイトル（クリックでQiita元記事へ遷移） |
| 著者 | アイコン画像 + ユーザー名 |
| 投稿日 | YYYY/MM/DD 形式 |
| いいね数 | ハートアイコン + 数値 |
| タグ | 最大5件のタグバッジ |

---

### データソース
- **Qiita**: Qiita API v2 を直接使用
- **Zenn**: Zenn公開API (`https://zenn.dev/api/articles`) を使用

### ソースフィルター
- すべて（Qiita + Zenn の記事を統合表示）
- Qiitaのみ
- Zennのみ

### フィルタリング機能
- タグ別フィルター（複数選択可）
- 対象タグ: `AI`, `機械学習`, `LLM`, `ChatGPT`, `DeepLearning`, `OpenAI`, `RAG`, `生成AI`
- 未選択時はすべてのAI関連記事を表示
- Zennはタグに対応するトピック名で検索

### ソート機能
- 新着順（デフォルト）
- いいね数順

### ページネーション
- 1ページ20件
- 「前へ」「次へ」ボタン

---

### 非機能要件
- レスポンシブ対応（ブレークポイント: sm=640px, md=768px, lg=1024px）
- ダークモード対応（OSの設定に追従）
- ローディング中はスケルトンUIを表示
- APIエラー時はエラーメッセージを表示（リトライボタン付き）
